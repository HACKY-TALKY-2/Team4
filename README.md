# 역3역 :3
흩어져있는 3개의 지하철 역을 누구보다 빠르게 찾아내자!

## 기능

### 1. 역3역 게임
- 격자판에 흩어져있는 3개의 역 이름을 맞춥니다.

### 2. 리더보드
- 게임을 한 player들의 순위를 확인할 수 있어요.
- 1등부터 10등까지의 기록을 확인할 수 있어요.
- 게임한 round가 많을수록, play한 round의 시간이 짧을수록 순위가 높아요.

### 3. 도움말
- 역3역 게임을 어떻게 하는지 확인할 수 있어요.

## 기술 스택
- BE : Node.js express
- FE : React
- DB : MySQL
